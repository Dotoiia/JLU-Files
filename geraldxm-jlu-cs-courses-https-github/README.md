# JLU-CS-Course-Notes

这是一个用于整理吉林大学计算机相关课程资料的开源仓库模板。目标是把课程笔记、实验说明、复习提纲、经验总结等内容沉淀下来，方便同学检索、补充和维护。

> 本仓库不直接复制其他仓库内容。建议只提交你拥有权利、自己整理、或明确允许公开分发的资料。

## 推荐资料类型

- 课程笔记：Markdown、PDF 或导出的 OneNote 笔记。
- 实验资料：实验要求、环境配置、踩坑记录、非敏感示例代码。
- 复习资料：知识点索引、题型总结、错题整理。
- 经验分享：选课建议、考试范围回忆、学习路线。

## 不建议提交

- 受版权保护且未授权公开传播的课件、教材扫描件、试卷原件。
- 可能违反课程要求的完整作业答案、实验报告成品。
- 带有个人隐私、学号、账号、密钥或内部系统截图的文件。

## 如何贡献

1. Fork 本仓库。
2. 在对应课程目录下添加资料。
3. 按照 `CONTRIBUTING.md` 中的格式补充说明。
4. 发起 Pull Request。

## 资料命名建议

```text
课程名/
  README.md
  资料/
    2026-复习提纲.md
    第01章-绪论.md
  作业与实验/
    lab1-环境配置.md
```

文件名尽量包含年份、主题或章节，方便后来的人判断适用范围。

## 免责声明

仓库内容由贡献者整理，仅供学习交流。资料可能随课程安排变化而过期，请以任课教师和课程平台发布的信息为准。
